# NeuroCashEngine - PostgreSQL 18 升级决策记录

**日期**: 2026-05-08  
**升级版本**: PostgreSQL 16 → PostgreSQL 18

---

## 🎯 升级决策原因

### 1️⃣ 与其他产品统一版本
- 其他产品将使用PostgreSQL 18进行RAG和语义向量检索
- 统一版本降低运维成本，避免碎片化
- 团队经验可以共享

### 2️⃣ 更好的语义向量检索支持
- PostgreSQL 18对向量检索有更好的性能优化
- 查询计划更高效
- 对pgvector扩展有更好的原生支持

### 3️⃣ 更长的支持生命周期
- PostgreSQL 18支持到2030年11月
- 相比PostgreSQL 16的2028年11月，多了2年保障
- 更适合长期产品
