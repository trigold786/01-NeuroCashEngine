# 商业需求文档 (BRD)：现金增值引擎 (NeuroCashEngine) v0.2.0

## 1. 宏观背景与项目执行摘要
本项目旨在抓住中国下沉市场财富觉醒与小微企业数字化转型的历史交汇点，打造一个“轻量级、自动化、强合规”的金融服务生态。v0.2.0 版本在 v0.1.0 的基础上，进一步聚焦并深化了 C 端用户的基础免费功能与增值服务、B 端用户的现金流与投资策略管理增值服务，以及基于脱敏用户数据的数据 API 产品，旨在提供更精准、更具价值的金融解决方案。

对于产品经理而言，本项目的核心交付标准是：**将极度复杂的机构级金融逻辑（如量化交易、企业现金流推演），降维封装为极其简单的用户体验。** 我们不仅提供财务诊断，更提供直接可执行的交易策略与标准化操作程序（SOP），彻底打通从“战略规划”到“战术落地”的最后一公里。

## 2. 市场机会与用户痛点深度剖析 (Why & Who)

### 2.1 C端（个人投资者）：从“盲目投机”到“纪律化投资”的鸿沟
*   **痛点一：资产配置（SAA/TAA）的认知真空。** 绝大多数散户的资金呈极度碎片化分布，缺乏穿透经济周期的全局大类资产（股/债/金/现）配置视角，极易在单一赛道（如全仓单只股票）承受致命风险。
*   **痛点二：高阶量化交易的极高门槛。** 市场上缺乏面向个人的、能深度整合沪深两市（上交所、深交所）双交易所账户的自动化交易工具。散户难以克服人性弱点（追涨杀跌），缺乏将个人交易理念转化为可自动执行的量化规则模块的能力。
*   **痛点三：信息过载与真伪难辨。** 个人投资者面临海量信息，难以有效筛选、辨别官方与非官方信息的真实性与时效性，导致决策失误。

### 2.2 B端（小微企业主）：资金链的“阿喀琉斯之踵”
*   **痛点一：公私账户混同与现金流盲区。** 企业经营资金与老板个人家庭资金深度绑定，缺乏跨银行的多账户聚合视图，导致老板对真实的可用流动性缺乏前瞻性预判。
*   **痛点二：财务调拨高度依赖“人治”，缺乏SOP。** 小微企业通常无力聘请专业CFO，多由缺乏财务深度的文员或行政人员兼任出纳。一旦面临复杂的税期准备或突发性大额支出，极易因缺乏标准化、傻瓜式的操作指南（SOP）而导致操作失误或资金链断裂。
*   **痛点三：投资策略的缺失与低效。** 小微企业主往往专注于主营业务，缺乏专业的投资知识和时间进行有效的资金增值管理，导致企业闲置资金收益率低下或面临不必要的风险。

### 2.3 数据需求方（宏观投研/量化机构）：微观高频数据的渴求
*   **痛点**：市场上充斥着宏观经济指标，但极度缺乏真实、连续、高频的“零售端投资情绪热力”与“小微实体资金周转率”等多维底层脱敏数据，这些正是优化量化模型的核心因子。

## 3. 产品战略与核心护城河 (What & How)

### 3.1 C端战略：机构级量化能力的“平民化”与个性化信息服务

#### 3.1.1 基础免费功能
*   **四大基本产品投资比例展示**：为用户提供其在现金、存款、基金、股票这四大基本金融产品上的实时投资比例概览。系统将通过用户授权的账户数据，自动聚合并可视化展示其在这些资产类别中的配置比例，帮助用户初步了解自身资产结构。
*   **官方渠道时事、财经资讯**：聚合来自央媒（如新华社、人民日报、CCTV财经）、证监会、银保监会等官方权威渠道的最新时事与财经新闻。确保信息的权威性、准确性与合规性，为用户提供可靠的市场动态参考。

#### 3.1.2 增值服务功能
*   **多渠道交叉验证的最新时事、财经资讯**：在官方资讯基础上，进一步整合非官方但经过多渠道（如知名财经媒体、独立研究机构、行业专家分析等）交叉验证的深度时事与财经解读。这些信息将提供更广阔的视角和更深入的分析，帮助用户理解市场背后的逻辑，但会明确标注其非官方属性。
*   **高级产品投资比例组合及具体投资标的、策略与交易方案**：
    *   **高级产品覆盖**：扩展至债券、贵金属（如黄金、白银）、期货、REITS（房地产投资信托基金）等多元化高级金融产品。
    *   **个性化投资组合**：基于用户的风险偏好、资金量、投资期限等因素，系统将智能生成包含上述高级产品的定制化投资比例组合建议。
    *   **具体投资标的推荐**：针对每个高级产品类别，提供经过深度研究和筛选的具体投资标的（如特定债券代码、黄金ETF、热门REITS项目等）。
    *   **详细投资策略**：为每个推荐标的提供详细的投资策略，包括但不限于入场时机、持仓周期、止盈止损点、风险管理建议等。
    *   **可执行交易方案**：将投资策略转化为具体、可操作的交易方案，指导用户如何通过现有交易渠道进行买卖操作，降低执行门槛。

### 3.2 B端战略：化身小微企业的“AI 虚拟 CFO”

#### 3.2.1 现金流管理
*   **多账户聚合与实时视图**：系统将通过用户授权，聚合企业在不同银行、不同账户（对公、对私、支付宝、微信支付等）的资金流水，提供实时的、穿透式的现金流总览视图。
*   **智能预测与预警**：基于至少3个月的历史流水数据，结合企业经营规律（如税期、发薪日、大额应收应付），精准预测未来3-6个月的现金流低谷与高峰。系统将设置硬性阈值，提供7x24小时的资金缺口或盈余告警，并给出初步的应对建议。
*   **资金调拨SOP生成**：当预测到资金缺口或盈余时，系统将自动生成详细、无技术门槛的标准化操作程序（SOP）文档，指导企业出纳或行政人员进行资金调拨（如提前赎回理财、短期借款、闲置资金归集等），确保操作的规范性与及时性。

#### 3.2.2 无保留的投资及策略管理
*   **企业闲置资金投资建议**：针对企业账户中的闲置资金，系统将根据企业的风险承受能力、资金使用周期等因素，提供定制化的投资建议，涵盖短期理财、结构性存款、低风险债券等，旨在在保证资金安全的前提下实现最大化增值。
*   **策略生成与执行指导**：为企业提供可执行的投资策略，包括但不限于资金配置比例、具体金融产品选择、风险控制措施等。系统将指导企业用户如何将这些策略应用于实际操作，并提供必要的交易方案。
*   **投资组合监控与调整**：持续监控企业投资组合的表现，并根据市场变化、企业资金状况等因素，及时提供调整建议，确保投资策略的有效性和适应性。

### 3.3 数据战略：合规的数据炼金术与API产品化
*   **脱敏与API集市**：在严格遵守k-匿名性与l-多样性的前提下，将沉淀的C端行为数据（如投资偏好、产品关注度、资讯阅读习惯）与B端现金流数据（如资金周转率、行业支付链特征）进行深度脱敏、聚合与分析。
*   **数据产品API化**：将脱敏后的数据加工成若干个标准化的API产品，面向宏观投研机构、量化基金、商业银行等B端客户提供订阅服务。这些API将提供微观、高频、多维度的数据洞察，例如：
    *   **C端投资情绪指数API**：反映不同资产类别（股、债、金、现）的散户情绪热力与资金流向。
    *   **小微企业资金周转率API**：提供特定行业或区域小微企业的平均资金周转速度与流动性健康度。
    *   **区域消费活力指数API**：结合C端与B端数据，反映特定区域的消费活跃度与经济景气度。
    *   **产品偏好与风险画像API**：提供匿名化的用户投资产品偏好分布与风险画像特征。

## 4. 商业化逻辑与增长引擎 (Business Model)

产品采用经典的“三级火箭”变现模型，辅以极具侵略性的裂变机制，并新增数据产品订阅服务：

### 4.1 基础引流层（获客）
*   免费提供基础的C端财务体检报告与B端单账户流水可视化分析，最大限度降低用户的体验门槛，建立初始流量池。
*   免费提供 C 端四大基本产品投资比例展示和官方渠道时事、财经资讯，吸引并留存大量基础用户。

### 4.2 核心变现与裂变层（LTV挖掘）
*   **C端增值服务订阅**：针对 C 端增值服务（多渠道交叉验证资讯、高级产品投资组合及策略交易方案），采用订阅制或按次付费模式，提供不同层级的服务包。
*   **B端增值服务订阅**：针对 B 端现金流管理与无保留的投资及策略管理服务，采用企业级订阅模式，根据服务深度和企业规模收取年费或月费。
*   **积分驱动的自发裂变**：建立“1积分=1元”的硬通货体系（设每月抵扣上限¥100）。赋予每个用户生成专属推荐链接的能力，被推荐人产生购买后，推荐人可获得高达20%-30%的积分返佣。以极其可观的利益驱动现有用户自发进行社交裂变，将获客成本（CAC）降至最低。

### 4.3 数据溢价层（利润放大器）
*   **数据产品API订阅服务**：面向机构销售脱敏后的 C 端行为数据与 B 端现金流数据加工而成的 API 产品订阅服务。这部分边际成本极低，是系统后期的核心利润池，提供按调用量、数据维度或订阅周期计费的灵活方案。

## 5. 赋能PM：产品演进路线图 (Roadmap)

为适配AI开发团队3天的极限交付，产品经理在拆解PRD时需严格遵循以下阶段：

*   **Phase 1 (Day 1) - 数据基座与体检引擎**：重点跑通全局用户ID映射（Hash加密）、事件埋点架构，以及C端的风险画像算法和配置模型。实现 C 端四大基本产品投资比例展示和官方渠道时事、财经资讯的聚合与呈现。
*   **Phase 2 (Day 2) - 量化战术与双所对接**：重点攻克金融数据API（AKShare/EastMoney）的对接，实现A股个股评级算法、沪深双账户整合逻辑，以及7x24小时的后台风控预警服务。实现 C 端增值服务中的多渠道交叉验证资讯和高级产品投资组合、策略与交易方案的初步功能。
*   **Phase 3 (Day 3) - B端SOP、数据API与商业闭环**：重点实现现金流推演算法、将结果转化为无门槛文员级SOP的生成器，并彻底打通微信/支付宝支付网关与裂变积分结算中枢。同时，完成 B 端无保留的投资及策略管理模块的开发，并初步构建脱敏数据 API 产品的接口与文档，为机构客户提供数据服务。
